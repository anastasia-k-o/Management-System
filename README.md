# 🏛️ ZIL Cultural Center Management System

**ZIL Cultural Center Management System** — это десктопное приложение для полной автоматизации деятельности Культурного центра ЗИЛ. Система охватывает три ключевых направления: развлекательное, образовательное и культурно-просветительское. Разработано на Python с использованием PyQt5 и SQLAlchemy, обеспечивает удобное управление пространствами, мероприятиями, студиями, выставками, продажами билетов и абонементов, а также полный документооборот выставок.

## ✨ Возможности

### Развлекательная деятельность
- Управление пространствами (залы, студии) с детализацией по локациям, рядам и местам.
- Регистрация мероприятий с автоматическим подбором пространства по вместимости.
- Интерактивная установка цен на билеты: выделение групп мест мышкой и назначение стоимости.
- Продажа билетов с визуализацией занятых/свободных мест.
- Отчёты по продажам за период с детализацией по ценовым категориям.

### Образовательная деятельность
- Ведение студий, преподавателей и посетителей.
- Создание приказов о работе студий с проверкой занятости преподавателя.
- Регистрация заявок на посещение студий.
- Установка цен на абонементы (разовый, месяц, год) с автоматическим расчётом стоимости.
- Продажа абонементов с подстановкой цены.
- Отчёты по набору в студии и продажам абонементов.

### Культурно-просветительская деятельность (Выставки)
- Учёт экспонатов с владельцами (Культурный центр, студия, сторонняя организация).
- Создание выставок (внутренняя/внешняя) и приказов о проведении.
- Автоматический отбор экспонатов по типу выставки.
- Оформление поступления экспонатов от сторонних организаций, актов передачи и возврата.
- Контроль этапов организации выставки через «карту маршрута» с блокировкой пропусков.

### Общее
- Загрузка начальных данных из Excel (пространства, студии, экспонаты, преподаватели).
- Гибкая настройка подключения к БД (SQLite по умолчанию, поддержка PostgreSQL).

## 🛠 Технологии

- **Язык**: Python 3.9+
- **Графический интерфейс**: PyQt5
- **ORM**: SQLAlchemy (Core + ORM)
- **База данных**: SQLite / PostgreSQL
- **Работа с Excel**: openpyxl, pandas
- **Отчёты**: ReportLab (PDF), matplotlib
- **Визуализация схем залов**: кастомные виджеты PyQt на QGraphicsScene
